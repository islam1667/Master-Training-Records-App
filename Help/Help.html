Help - Master Training Records

Database format

Database must be formatted like this, else it will cause errors and will not work properly.

How to select database of trainings

Once the database is choosen, next time when app launched it will load it from the file you chose previously. You can change database file by clicking DB button.

How to use

Grid and the list works in sync. You choose the trainings from the list which you want to add.

How to save

You can save your file by clicking the save button located in File>Save or by clicking Ctrl+S.

How to Select cells

Single click to select cell. Selecting and typing in the keyboard will edit the cell.

Ctrl + select for multi select.

Shift + select for range select.

How to reset table or textBoxes

Clicking appropriate reset button will reset textBoxes or the grid.

How to delete rows

You can delete rows by selecting row from header.

Single click to select cell. Selecting and typing in the keyboard will edit the cell.

Ctrl + select for multi select.

Shift + select for range select.

Date formats

For StartTime and EndTime columns you can easily write dates in the formats listed below.

'dd MM yyyy'
'dd MM yy'
'd MM yyyy'
'd MM yy'
'dd M yyyy'
'dd M yy'
'd M yyyy'
'd M yy'

How to print

You can print by clicking the print button located in File>Print.

You dont need to save document to print.

Excel printing window will open to show preview and print.
